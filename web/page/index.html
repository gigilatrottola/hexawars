HexaWars is a turn-based multiplayer strategy game for mobile phones. The goal of the game is to occupy the forts with your armies. The game can be played by two to six players depending on the game type. All of the messages goes to the game server, a GPRS connection is needed to play the game. One average game lasts 10 to 20 minutes and transfers 20-30 kByte data.

If there are no other human players at a certain timeframe the game will be started with artificial opponents. The artificial opponents are on different level.

Technical details.
Currently the game is available for Nokia phones with screen size 128x128 and 176x208 and for the Nokia API compatible devices like Sony Ericsson K300/J300 and K700/K750. The midlet uses MIDP-1.0 profile and CLDC-1.0 configuration.

The game server can handle 1800 messages per second. With the average message sending profile it can serve 40000 logged users paralelly. If this number wouldn't be enough there is a redirect mechanism for load balancing purposes. The performance can be doubled, tripled, etc. with adding new hardware. HexaWars is the first game developed for this architecture. The developement of the second and third game is on the way. Those will be integrated to the same server.
